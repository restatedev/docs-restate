# Durable Workflows

Create robust workflows that handle failures gracefully and maintain consistency across distributed systems.

## What are Durable Workflows?

Durable workflows are long-running processes that maintain their state and progress even in the face of failures, network issues, or system restarts.

## Key Benefits

- **Automatic state management**: No need to manually persist workflow state
- **Built-in retries**: Failed steps are automatically retried with exponential backoff
- **Compensation logic**: Easy rollback and compensation for failed workflows
- **Visual debugging**: Track workflow execution and state changes

## Use Cases

- Order processing pipelines
- Data processing workflows
- Multi-step approval processes
- Distributed transactions
