[Install Restate](/installation) and launch it:
```bash
npm install --global @restatedev/restate-server@latest @restatedev/restate@latest
restate-server
```

Get the example:
```bash
git clone https://github.com/restatedev/ai-examples.git && cd ai-examples/typescript-patterns
npm install
```

Export your API key and run the service:
```bash
export OPENAI_API_KEY=sk-...
npm run dev
```

Register the services with Restate:
```bash
restate deployments register http://localhost:9080
```
