---
title: Context Helpers
description: Built-in tools every handler gets
---

Every handler gets a `ctx` parameter with helpful tools. These handle the tricky parts of building reliable services so you don't have to.

### **Essential Helpers** – core execution patterns

Use `ctx.run()` to wrap database calls and API requests safely. Call other services reliably with automatic retries and delivery guarantees.

### **State & Timing Helpers** – data and time management

Store and retrieve data with strong consistency. Use `ctx.sleep()` to pause execution or wait for external events. All operations survive crashes and restarts.

Save and load data with `ctx.get()` and `ctx.set()`. Your data stays consistent and survives failures automatically.
