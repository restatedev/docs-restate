## Aborting execution

Cancellation can be done over http

```shell
curl -X PATCH http://localhost:9070/invocations/inv_1gdJBtdVEcM942bjcDmb1c1khoaJe11Hbz/cancel
```

If the behavior should be more advanced than full cancellation, then you need to implement it on the agent side.
See the other guide for more information

Also have a look at the rollback guide to implement reverting critical actions.

## Regenerating responses

Regenerate can be done via Restate's restart-as-new feature:

```shell
curl -X PATCH http://localhost:9070/invocations/inv_1gdJBtdVEcM942bjcDmb1c1khoaJe11Hbz/restart-as-new
```
