LiteLLM is a library that simplifies multiple model providers.
You can combine it with Restate to build resilient agents that can be customized to your use case.
You [wrap model calls in `ctx.run`](/ai/sdk-integrations/integration-guide#integrating-ai-sdks) and use Restate context actions to execute tools.

<Card icon="/img/ai/sdk-integrations/lite-llm_icon.webp" title="Learn more about LiteLLM" horizontal href="https://www.litellm.ai/"/>

Have a look at the following resources to get started with Restate and LiteLLM:
- [Examples](https://github.com/restatedev/ai-examples/tree/ai-structure/python-restate-only/tour-of-agents)
- [Guides](/ai/patterns/durable-agents): Most patterns have been implemented using LiteLLM for model calls.
