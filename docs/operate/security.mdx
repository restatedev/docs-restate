---
title: Security
description: Secure your Restate deployments and protect your data
---

Control access to your Restate services with authentication tokens and role-based permissions. Configure secure communication between services and external systems.

## Network security

Set up secure network configurations, SSL/TLS encryption, and firewall rules to protect your Restate cluster from unauthorized access.

## Data protection

Encrypt sensitive data at rest and in transit. Configure backup and recovery procedures to protect against data loss.
